# Homebrew Tap

Homebrew tap for `stanislavkozlovski/dclaude`.

> 👉 https://github.com/stanislavkozlovski/dclaude/

Installs:

- `dclaude`
- `dcodex`

```bash
brew tap stanislavkozlovski/tap
brew install dclaude
```

You can also install directly without tapping first:

```bash
brew install stanislavkozlovski/tap/dclaude
```
